# Images Directory

Add your project images here.

## How to Use

1. Add your images to this folder
2. Update the paths in `index.html`

## Supported Formats

- JPEG (.jpg)
- PNG (.png)
- WebP (.webp)
- SVG (.svg)

## Image Sizes

Recommended dimensions:
- Project images: 400x300 or 800x600
- Profile photo: 400x400
- OG images: 1200x630 (for social media sharing)
